Building Blocks of a Website & How Websites Work

Websites are created using HTML, CSS, and JavaScript. HTML is the foundation—you can't create a website without mastering HTML first. CSS and JavaScript add styling and interactivity.

When you click on text on one page and it redirects you to another, that text is called a hypertext or hyperlink. This concept was introduced in the world's first website created by Tim Berners-Lee.

Front-end developers create static websites, and combining front-end, back-end, and databases creates a full-stack website.

HTML Text Formatting and Nesting

Example of proper nested tags:

My cat is very grumpy.

Incorrect nesting causes errors:

<p>My cat is <strong>very grumpy.</p></strong>

Void (Empty) Elements

Void elements do not have closing tags. Examples include:

  • <img> for images
  • <input> for form inputs
  • <link> for stylesheets or icons
  • <hr> for horizontal lines
  • <br> for line breaks

HTML Anchor Tags & Links

Create links using the <a> tag with the href attribute. Example:

Visit Thapa Technical

Anchor Tag Tips

  • Internal page link: Use href="#sectionID" to jump within the same page.
  • Relative vs Absolute URLs: Relative links point to files within the site; absolute links include full URL.
  • Download attribute: <a href="file.pdf" download>Download PDF</a> forces file download.
  • Email link: <a href="mailto:email@example.com">Send Email</a>
  • Phone link: <a href="tel:+1234567890">Call Us</a>

Semantic HTML Tags

Semantic tags add meaning to your HTML, improving accessibility and SEO. Common semantic tags:

  • <header> – Page or section header
  • <nav> – Navigation links
  • <main> – Main content
  • <article> – Self-contained content
  • <section> – Thematic grouping
  • <footer> – Page or section footer
  • <aside> – Sidebar content
  • <figure> and <figcaption> – For images with captions
