🔍 Anchor Tag Attributes

  • href: URL the link points to.
  • target: Where to open the link (e.g., _blank).
  • download: Initiates download of the linked file.
  • title: Tooltip text when hovering.
  • id: Unique identifier for internal linking.
  • class: For CSS/JS styling or targeting.
  • style: Inline styling.
  • tabindex: Keyboard navigation order.
  • accesskey: Keyboard shortcut.
  • aria-*: For accessibility.

📘 Anchor Tag Interview Questions

  1. How to create a link within the same page?
    Use <a href="#section"> and the section must have id="section".
  2. Difference between Relative and Absolute URLs?
    Relative: about.html
    Absolute: https://example.com/about.html
  3. How does download attribute work?
    Suggests the browser to download the file instead of navigating to it.
  4. How to create an email link?
    <a href="mailto:someone@example.com">
  5. How to initiate a phone call?
    <a href="tel:+1234567890">

🎁 Bonus Tips

  • ✅ Always use quotes in attributes: <a href="https://...">
  • ✅ Single or double quotes are both fine, but don’t mix!
  • ✅ You can use one type of quote inside the other: <a href="..." title="Isn't this cool?">

💡 HTML Entities

HTML entities represent special characters. Useful when characters conflict with HTML or are not available on keyboard.

  • < 👉 Less than sign (<)
  • > 👉 Greater than sign (>)
  • & 👉 Ampersand (&)
  • " 👉 Double quote (")
  • ' 👉 Single quote (')
  •   👉 Non-breaking space
  • © 👉 ©
  • ® 👉 ®
  • ™ 👉 ™
  • ♥ 👉 ♥
  • $ 👉 $

    To prevent collapsing of multiple spaces.
  2. How to show trademark symbol?
    Use &trade; = ™

    To avoid syntax conflicts and ensure proper display.
